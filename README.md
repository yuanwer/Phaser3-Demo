### 02_吃星星
Phaser官网教程，制作第一个游戏

- 玩家通过方向键 `↑↓←→` 控制小人儿
- 吃掉 `⭐` 会增加得分
- 每吃完一轮 `⭐` ，会刷新下一轮 `⭐` ，同时投放一枚 `💣`
- 小人儿碰到 `💣` ，游戏结束

![游戏截图](https://raw.githubusercontent.com/Yaper21/picGo/main/img/image-20220828151712380.png)
